Third Party Rhythmbox Plugins

This page gives links to plugins developed outside of the Rhythmbox project itself, by developers who are generally not directly involved in the project. Users should be careful when installing third party plugins such as these as there are no restrictions on what plugins can do.

If you've written a plugin that you'd like to have added to this list, submit a merge request in GitLab that adds your .plugin file under third-party-plugins in the website branch. Entries in the list can also be updated via a merge request.
